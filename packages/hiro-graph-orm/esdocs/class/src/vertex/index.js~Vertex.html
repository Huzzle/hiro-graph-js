    public setVertices(relation: string, nodes: Array<Vertex>): Vertex source

    Set the relationship data for a vertex (with related vertices)

    This may seem superfluous because we don't set the nodes, just the ids. but actually we don't want the interconnections. the GraphVertex can pull the relations from cache, setting this here with the different key is saying "I fetched the nodes".
