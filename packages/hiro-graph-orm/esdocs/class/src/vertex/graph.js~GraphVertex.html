    GraphVertex

    Extends:

    Vertex → GraphVertex

    This is like the regular vertex, but has database access. Depending on your use case, you would do different things. If your frontend has no direct GraphIT access, the use DataVertex If you are on the backend (or frontend with graph access) use the GraphVertex which can be saved.
