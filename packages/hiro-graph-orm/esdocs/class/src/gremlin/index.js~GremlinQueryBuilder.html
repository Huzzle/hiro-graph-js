    GremlinQueryBuilder

    A simple gremlin query builder object.

    Provides a convenience for having to produce the queries by hand, and escapes the terms correctly.

    public append(query: *): GremlinQueryBuilder source

    Append to the last chunk in the stack,

    This is used for the very rare instances where you need to add directly to the last element in the stack rather than adding a new element (which would be . seperated)

    The example is a limit clause which appends [0..1] directly to the current pipeline.
